Contests-Engine

This URL provides access to the public API of the Contests-Engine, a tournament management back-end system that handles contests, participants, draws, matches and the like for almost any contest (football championships, tennis trophies, billard tournaments and so on).

The goal of the Contests-Engine is to let developers of tournament management front-ends (websites, mobile apps, desktop apps, ...) concentrate on the presentation layer of their systems, e.g.

  • Fast and simple data entry forms
  • Easy definition of a draw by dragging around participants on the empty tableau
  • Elegant renderings of large groups- or knock-out-tableaus with scrolling and zooming
and so on. The Contest-Engine in the background then handles tasks as
  • Store, update and retrieve all data of the contests
  • Randomly fill an incomplete draw or create an entire draw with due regard to special rules like seed lists
  • Create the matches for each round according to the type of the contest (round robin, groups, elimination or double-elimination game, ...)
  • Keep everything actualized whenever a new result is received, e.g. propagate the winner of a elimination game to the next round or group winners to the final group or elimination round
  • Distribute points for wins, losses and eventually ties, calculate rankings

Further Details can be found at the Contests-Engine Github page.
