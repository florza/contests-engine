iTurnier-Backend

This URL provides access to the backend of the iTurnier II application.

It does not provide its own web pages and descriptions, but is accessed from frontend applications over an API, with REST services to:

  • create a contest and participants
  • create a draw
  • create the contest tableau with all the matches
  • edit match results and update the tableau

The application is still under development. For details, send a mail to rolf.zanga@icloud.com.
